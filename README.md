## 🧠 Model Details

- Trained on Fashion-MNIST dataset (28x28 grayscale images)  
- Input → 28x28 image  
- Output → 10 classes:
  - T-shirt/top 👕
  - Trouser 👖
  - Pullover 🧥
  - Dress 👗
  - Coat 🧥
  - Sandal 👡
  - Shirt 👔
  - Sneaker 👟
  - Bag 👜
  - Ankle boot 👢

- ANN Architecture:
  - Flatten layer → Dense layers with ReLU → Dropout → Softmax output

---

## ✅ Features

- Draw fashion item on canvas
- Predict item in real-time
- Display confidence score
- **Educational Version**: ANN may give incorrect predictions sometimes
